# W-Wave-Radio

## Get started

```javaScript
npm i
```

Dev:

```javaScript
npm run start
```

Dev-build (no minify):

```javaScript
npm run dev
```

Production:

```javaScript
npm run build
```

## APIs

- [Choices](https://github.com/Choices-js/Choices)
- [Accordion](https://github.com/michu2k/Accordion)
- [Swiper](https://swiperjs.com/)
